## เรื่อง Paragraph Structure (16DEE251)

```xml
<w:p w14:paraId="16DEE251" ...>
  <w:pPr>
    <w:ind w:left="567" w:hanging="567"/>
    <w:jc w:val="thaiDistribute"/>
    ...
  </w:pPr>
  <!-- decorative line shape (AlternateContent) -->
  <w:r>...<w:drawing>...</w:drawing></w:r>
  <!-- label -->
  <w:r><w:rPr><w:b/><w:sz w:val="40"/>...</w:rPr><w:t>เรื่อง</w:t></w:r>
  <!-- spacing run -->
  <w:r><w:rPr>...</w:rPr><w:t xml:space="preserve">  </w:t></w:r>
  <!-- ← REPLACE EVERYTHING AFTER HERE WITH SUBJECT TEXT -->
  <w:r>...<w:t>SUBJECT TEXT</w:t></w:r>
</w:p>
```

`replace_subject()` finds the run containing `<w:t>เรื่อง</w:t>`, locates the
closing `</w:r>` of that run, then replaces everything until `</w:p>` with the
new subject text run.

## Body Replacement Range

`replace_body()` replaces paragraphs from **4168ED9B** (inclusive) through the
paragraph immediately before **56B6A4BF** (exclusive).

The replacement XML is generated by `build_body()` which accepts a list of
block descriptors. Each block emits one or more `<w:p>` elements.

### Body XML pattern

All body paragraphs use this pPr:
```xml
<w:pPr>
  <w:ind w:right="-46"/>
  <w:jc w:val="thaiDistribute"/>
  <w:rPr>
    <w:rFonts w:ascii="TH Sarabun New" w:hAnsi="TH Sarabun New" w:cs="TH Sarabun New"/>
  </w:rPr>
</w:pPr>
```

### Indented paragraph (indent=true)
```xml
<w:r>RPR<w:tab/><w:tab/></w:r>
<w:r>RPR<w:t>TEXT</w:t></w:r>
```

### Numbered list item
```xml
<w:r>RPR<w:tab/><w:tab/></w:r>
<w:r>RPR<w:t>N. item text</w:t></w:r>
```

### Detail list item (label: value)
```xml
<w:r>RPR<w:tab/><w:tab/></w:r>
<w:r>RPR<w:t>        - </w:t></w:r>
<w:r>RPR_BOLD<w:t>label  </w:t></w:r>
<w:r>RPR<w:t>value text</w:t></w:r>
```

| paraId      | เนื้อหา                                               |
|-------------|------------------------------------------------------|
| `3289E0CF`  | `ลงชื่อ .................................................` (tabs + text) |
| `6E823166`  | ตำแหน่ง line ← replaced by `replace_sig_title()`     |
| `13AE0BD8`  | blank                                                |
| `170925A1`  | blank                                                |
| `54F541E4`  | blank                                                |

The signature name `(ผู้ช่วยศาสตราจารย์ ดร.รุสลี่ สุทธวีร์กูล)` is **hardcoded**
in the template and is **not replaced by the script**. Edit manually if needed,
or extend `generate()` with a `replace_sig_name()` function using the same
pattern as `replace_sig_title()`.

## ที่ / วันที่ Row (718F05D0)

The paragraph contains two decorative line shapes (AlternateContent) plus text runs:

- **เลขที่หนังสือ**: text run containing `วฟ         /2566`  
  → replaced by `replace_doc_number()` using regex `วฟ\s+/\d+`

- **วันที่**: split across three runs: `' 16'` + `' มิถุนายน'` + `'2566'`  
  → `replace_date()` replaces only the first `<w:t xml:space="preserve"> \d+</w:t>` (day number).  
  Month and year should be updated manually or by extending the function.

## Config JSON Reference

```json
{
  "เรื่อง": "ขออนุมัติ...",
  "ที่": "อว ๗๑๕๖.๐๑/___",
  "วันที่": "19",
  "ลงนาม": {
    "ชื่อ": "ผู้ช่วยศาสตราจารย์ ดร.รุสลี่ สุทธวีร์กูล",
    "ตำแหน่ง": "อาจารย์ประจำภาควิชาวิศวกรรมไฟฟ้าและคอมพิวเตอร์"
  },
  "เนื้อหา": [
    {"type": "paragraph",     "indent": true,  "text": "ด้วย..."},
    {"type": "list_intro",    "indent": true,  "text": "...มีดังนี้"},
    {"type": "numbered_list", "items": ["item1", "item2"]},
    {"type": "detail_list",   "items": [
      {"label": "ชื่อโครงการ", "value": "..."},
      {"label": "งบประมาณ",    "value": "50,000 บาท"}
    ]},
    {"type": "paragraph",     "indent": true,  "text": "จึงเรียนมา..."}
  ],
  "เอกสารแนบ": ["เอกสาร 1", "เอกสาร 2"]
}
```

Block types summary:
| type           | เนื้อหา                              |
|----------------|--------------------------------------|
| `paragraph`    | ย่อหน้าธรรมดา (indent = เว้นวรรค)  |
| `list_intro`   | ย่อหน้านำ list (ไม่มี blank หลัง)  |
| `numbered_list`| รายการ 1. 2. 3. ...                 |
| `detail_list`  | - **label**  value                   |
